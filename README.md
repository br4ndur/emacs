# Installation

Clone the repository:

    git clone git@github.com:branduren/emacs.git ~/dotfiles/emacs

Update submodules:

    cd ~/dotfiles/emacs
    git submodule init
    git submodule update submodule/nxhtml


Remember to set symlink:

    set symlink: ln -s ~/dotfiles/emacs/ ~/.emacs.d
